# 迷你网络群聊

上班摸鱼的网络聊天室

## 代做功能
### 基础功能

- [ ] 用户登录(目前可以采用springSecurity表单注册登录,后续可以考虑其他登陆方式)
- [ ] webSocket群内消息推送(基础能力)
- [ ] 群成员相关接口(成员列表/在线人数..)
- [ ] 消息相关接口(历史消息/)
- [ ] 个人信息相关接口(个人信息修改/信息查看/头像更改..)
### 上层功能
- [ ] 敏感词过滤
- [ ] URL识别
- [ ] @功能
- [ ] 多元消息发送(emoji/表情包/图片/文件/)
- [ ] ....


## 分支管理
